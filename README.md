# e-Canteen App

This Android app helps users book food items (Breakfast, Lunch, Snacks, Special Items) from the college canteen and allows the admin to manage orders.

## Features
- User authentication and profile
- Admin login panel
- Item booking system with real-time Firebase backend
- Category-based items: Breakfast, Lunch, Snacks, and Special Items
- Daily update view
- Order management with Pending/Delivered status
- Special item management by admin
- Firebase Realtime Database integration
- Payment mode selection (Online/Offline)
- Restrictions per user:
  - One **Breakfast** booking per day
  - One **Lunch** booking per day
  - One **Snacks** booking per day
  - Multiple **Special Items** allowed

## Note
- Push notifications are **not implemented yet** (will be added in future).

## Setup Instructions
1. Clone the repository
2. Open project in Android Studio
3. Connect your Firebase project and replace `google-services.json`
4. Run the app on an emulator or physical device
